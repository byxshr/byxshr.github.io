np.power(x1,x2) 数组的元素分别求n次方，x2可以是数字，也可以是数组，但是x1和x2的列数要相同。
`

x1 = range(6)

x2 = [1.0, 2.0, 3.0, 3.0, 2.0, 1.0]
np.power(x1, x2)
array([ 0., 1., 8., 27., 16., 5.])

np.sum(a, axis=None, dtype=None, out=None, keepdims=np._NoValue) :
a是要进行加法运算的向量/数组/矩阵
axis的值可以为None,也可以为整数和元组
其形参的注释如下:
a : array_like
Elements to sum. (用于进行加法运算的数组形式的元素)
axis : None or int or tuple of ints, optional
Axis or axes along which a sum is performed. The default,
axis=None, will sum all of the elements of the input array.
(如果axis的为None,则将数组中的每一个元素都加起来)
If axis is negative it counts from the last to the first axis.
If axis is a tuple of ints, a sum is performed on all of the axes
specified in the tuple instead of a single axis or all the axes as
before.
当axis为0时,是压缩行,即将每一列的元素相加,将矩阵压缩为一行 当axis为1时,是压缩列,即将每一行的元素相加,将矩阵压缩为一列

np.dot(x1,x2) 返回的是两个数组的点积(dot product)
1.如果处理的是一维数组，则得到的是两数组的內积
2.如果是二维数组（矩阵）之间的运算，则得到的是矩阵积（mastrix product）。
3.dot()函数可以通过numpy库调用，也可以由数组实例对象进行调用。a.dot(b) 与 np.dot(a,b)效果相同。矩阵积计算不遵循交换律,np.dot(a,b) 和 np.dot(b,a) 得到的结果是不一样的。

np.mean() :计算矩阵均值

a = np.array([[1, 2], [3, 4]])
np.mean(a) # 将上面二维矩阵的每个元素相加除以元素个数（求平均数）

array([ 2., 3.])
np.mean(a, axis=1) # 计算每一行的均值

np.std() ：计算矩阵标准差

array([ 1., 1.])
np.std(a, axis=1) # 计算每一行的标准差
array([ 0.5, 0.5])
